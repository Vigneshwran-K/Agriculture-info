Smart Farming or 'Krishoker haat' provides a platform where 

- users can buy/sell there products
- users can rent/hire there equipments
- users can use crop recommendation system (using linear regression)

used technologies:
Frontend:
HTML, CSS, JavaScripts, Bootstrap 4 
Backend:
PHP, MySQL

for crop recommendation system:
Python 3
Flask web framework
modules like scikit-learn,pandas,pickle
it is recommended to use anaconda to run this model

and some of the dataset taken from this- 

https://github.com/omkarbuchade/Smart-agricultural-system
